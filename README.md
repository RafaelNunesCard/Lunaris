Lunaris é um projeto fullstack desenvolvido com foco na prática e evolução em
programação, abrangendo desde a construção de interfaces modernas com React e
Tailwind até o desenvolvimento de um backend estruturado, integração com banco
de dados e organização de código em nível mais profissional.

## 🚀 Objetivo

O objetivo do projeto é:

- Praticar desenvolvimento de interfaces
- Aprender e aplicar Tailwind CSS
- Evoluir lógica de programação
- Organizar código de forma mais profissional
```bash
lunaris/
├── backend/
├── frontend/
└── README.md
```

## 📚 O que já foi feito
